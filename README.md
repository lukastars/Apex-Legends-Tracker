## Apex-Legends-Tracker[Live Preview](https://hidden-basin-62213.herokuapp.com)
A Full-Stack web application that tracks Apex Legends players status.

## Technologies
### Front-End:
* Vue.js
* HTML/CSS/JavaScript
* Vue-router, Vue-toasted, Axios 
### Back-End:
* Node.js/Express.js
* Heroku, Postman 
### Tools
* VS Code, Git
